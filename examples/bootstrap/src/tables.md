# Tables

### table
#### `table.table`

A basic table.

```example.jade
.panel.panel-default
  table.table
    thead
      tr
        th #
        th First name
        th Last name
    tbody
      tr
        th 1
        td Louie
        td Armstrong
      tr
        th 2
        td Frank
        td Sinatra
      tr
        th 3
        td Ella
        td Fitzgerald
```
